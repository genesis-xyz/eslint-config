# @genesis-xyz/eslint-config

Genesis's eslint + prettier setup.

---

# Typescript

## Installation

```
yarn add @genesis-xyz/eslint-config eslint prettier typescript --dev
touch .eslintrc.js
```

### `.eslintrc.js`

```javascript
module.exports = {
  extends: ["@genesis-xyz"],
};
```

---

# Javascript

yarn add @genesis-xyz/eslint-config eslint prettier --dev
